# 快速开始指南

5 分钟快速上手使用飞书文档筛选条件获取插件。

## 🚀 三步开始

### 步骤 1: 安装依赖

```bash
npm install
```

### 步骤 2: 启动开发服务器

```bash
npm run dev
```

### 步骤 3: 打开浏览器

访问 `http://localhost:3000`

> ⚠️ **注意**：在本地浏览器中会显示错误提示，这是正常的。需要在飞书文档环境中才能正常工作。

---

## 📝 完整配置（生产环境）

### 1. 创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. **获取 `App ID`**（只需要这个！）
4. 申请权限：`bitable:app:readonly`

> ✅ **重要**：这个插件是客户端插件，**只需要 `App ID`**，不需要 `App Secret`！

### 2. 配置项目

编辑 `manifest.json`，只需要替换 `app_id`：

```json
{
  "app_id": "你的_app_id"  // 只需要替换这里
}
```

### 3. 部署

#### 使用 Vercel（推荐）

```bash
npm i -g vercel
vercel
```

#### 或使用其他平台

```bash
npm run build
npm start
```

### 4. 在飞书中使用

1. 在飞书开放平台配置插件 URL
2. 在飞书多维表格中插入网页应用
3. 授权并开始使用
